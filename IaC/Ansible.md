# Ansible

## Installation
Check python version
```sh
python --version
pip --version
```

Install ansible as python module
```sh
python3 -m pip install --user ansible
```

Upgrading ansible
```sh
python3 -m pip install --upgrade --user ansible
```

Verify
```sh
ansible --version
```
